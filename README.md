# Emotion-Ledger PWA

Installierbare iPhone-optimierte PWA für Time/Event Logging.

## Lokal starten

```bash
npm install
npm run dev
```

## Auf Vercel deployen

1. Ordner als neues Vercel-Projekt importieren oder hochladen.
2. Framework wird als Vite erkannt.
3. Deploy klicken.
4. Den Vercel-Link am iPhone in Safari öffnen.
5. Teilen → Zum Home-Bildschirm.

## Enthalten

- Emotion-Ledger Branding mit Logo
- Event-Typen: Denken, Fühlen, Planen, Musik, Location, Body
- Wallet-Ansicht
- Advanced Dashboard: Verteilung, Trend pro Tag, Peak Hours, Wochen-Heatmap, Live Timeline
- Lokale Speicherung im Browser
- CSV Export
- iPhone PWA Settings und Service Worker
